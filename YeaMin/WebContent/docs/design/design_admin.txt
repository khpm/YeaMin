로그인 화면
	- 회원 정보를 사용하여 로그인한다.
	- 로그인에 실패했을 경우, 경고 메시지를 출력한다.

마이 메이지 - 회원 정보 수정
	- 회원 정보를 표시하며 다음 항목을 수정할 수 있다.
	- 수정할 항목 : 비밀번호, 이메일, 핸드폰 번호

가게 관리 - 가게 정보 관리
	- 가게 정보를 등록, 수정할 수 있다.

가게 관리 - 상품 카테고리 정보 관리
	- 상품 카테고리 정보를 조회, 등록, 수정, 삭제할 수 있다.
	- 조회 : 상품 카테고리 정보와 각 상품 카테고리에 포함된 상품의 개수를 함께 표시한다.
	- 삭제 : 해당 상품 카테고리에 포함된 상품이 있으면 삭제가 불가하다.

가게 관리 - 상품 정보 관리
	- 상품 정보를 조회, 등록, 수정, 삭제할 수 있다.
	- 조회 : 상품 정보와 각 상품 정보에 해당하는 상품 카테고리 정보를 함께 표시한다.
	★★★★ 상품 정보 수정, 삭제 시 예약 정보의 데이터가 불안정하다. ★★★★

예약 관리 - 예약 현황 및 이력
	- 예약 정보를 조회, 상세보기, 취소할 수 있다.
	- 조회 : 사용자 정보, 예약 정보와 예약한 상품들의 합계 금액을 함께 표시한다.
	- 상세보기 : 사용자 정보, 예약 정보와 예약한 상품 정보와 상품들의 합계 금액을 합계를 표시한다.
	- 취소 : 현재 날짜가 예약 날짜보다 1일 전 일 경우 취소가 불가하다.

예약 관리 - 통계
	- 성별 리뷰 통계 : 남/녀에 대한 리뷰 수와 비율을 표시한다. 
	- 연령별 예약 수 통계 :20대, 30대, 40대, 50대, 기타에 대한 예약수를 표시한다.
	- 시간별 매출액 : 24시간을 2시간 단위로 매출액을 표시한다.
	- 시간별 예약 수 : 24시간을 2시간 단위로 예약수를 표시한다.
	- 상품 top 5 : 상품 카테고리, 특정 기간에 예약된 상품의 top 5 상품을 상품 수와 비율을 표시한다.

예약 관리 - 예약 시간 관리
	- 예약 시간 정보를 조회, 등록, 수정, 삭제할 수 있다.
	
회원 관리
	- 사용자의 정보를 표시한다.
	- 조회 : 사용자 정보로 조회한다.
