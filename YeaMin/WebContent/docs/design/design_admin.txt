F - C - 02 : 메인 화면
	<< 상단 >>
	- 로고는 메인 화면에 포함된다.(F - C - 01)
	- 로그인되지 않는 사용자라면 다음 기능을 할 수 있다.
		- 이메일, 비밀번호를 입력하여 로그인을 시도한다.
			- 로그인에 성공할 경우, 메인 화면으로 이동한다.(F - C - 02)
			- 로그인에 실패할 경우, 로그인 화면으로 이동한다.(F - C - 03)
		- 회원 가입 버튼을 통해 회원 가입 화면으로 이동한다.(F - C - 4)
	- 로그인된 사용자라면 다음 기능을 할 수 있다.
		- 로그아웃 버튼을 통해 로그아웃을 할 수 있으며 메인 화면으로 이동한다.(F - C - 02)
	- 고객센터 버튼을 통해 고객센터 화면으로 이동한다.(F - C - 17)

F - C - 03 : 로그인 화면
	- 이메일, 비밀번호를 입력하여 로그인을 시도한다.
		- 로그인에 실패할 경우, 다음과 같이 경우에 따라 메세지를 표시한다.
			- 이메일이 없는 경우, "입력하신 이메일/비밀번호가 잘못 되었습니다."라고 메시지 표시합니다.
			- 비밀번호가 일치하지 않는 경우, "입력하신 이메일/비밀번호가 잘못 되었습니다."라고 메시지 표시합니다.
		- 로그인에 성공할 경우, 로그인 화면에 이동되기 전 화면으로 이동한다.
		- 로그인에 성공할 경우, 확인 되지 않는 예약 건 수를 표시하며 "예약 확인"을 눌렀을 경우, 예약 현황 화면으로 이동한다.
	- 회원 가입 버튼을 통해 회원 가입 화면으로 이동 할 수 있다.(F - C - 4)

F - C - 04 : 회원가입 화면
	- 사용자 정보를 입력하여 회원가입을 합니다.
		- 사용자 정보 : 이메일, 비밀번호, 이름, 핸드폰 번호, 생년월일, 사업자등록번호
		  ※ 이메일 양식에 어긋나면 표시합니다.
		  ※ 이메일, 비밀번호, 핸드폰 번호, 생년월일, 사업자등록번호는 필수 항목입니다.
	- 사업자등록번호를 사용하여 중복 체크를 합니다.
	  ※ 해당 항목을 변경될 경우, 사업자등록번호 중복 체크와 사업자등록번호 인증을 다시 진행합니다.
	- 이메일을 사용하여 중복 체크를 합니다.
	  ※ 해당 항목을 변경될 경우, 이메일 중복 체크와 이메일 인증을 다시 진행합니다.
	- 이메일 중복이 되지 않았으면 이메일을 사용하여 인증 절차를 진행합니다.
	- 회원 가입 버튼을 누르면 이메일을 사용하여 중복 체크를 합니다.
	     중복이 되었다면 다시 입력하도록 합니다.

F - C - 06 : 가게 관리
	- 등록되어 있는 가게가 없다면 "가게 등록"을
	     등록되어 있는 가게가 있다면 "가게 수정"을 수행한다.
	- 가게 등록
		- 다음 항목을 등록한다.
			- 가게 이미지
			※ 가게 이미지는 하나만 설정 할 수 있다.
			- 가게 이름
			※ 가게 이름을 변경하면 중복 체크를 수행한다.
			- 가게 카테고리
			- 가게 전화 번호
			- 가게 주소
			- 가게 소개
			※ 300글자 이내로 작성 할 수 있다.
			- 엽업 시간(평일, 주말 - open, breack, close)
			- 휴무일
			- 테이블 수
		- 가게 등록을 할 경우, 가게 이름을 사용하여 중복 체크를 수행한다.
		- 가게 등록이 완료되면 "가게 관리 - 가게 수정"으로 이동한다.
	- 가게 수정
		- 다음 항목을 수정한다.
			- 가게 이미지
				※ 가게 이미지는 하나만 설정 할 수 있다.
			- 가게 이름
				※ 가게 이름을 변경하면 중복 체크를 수행한다.
			- 가게 카테고리
			- 가게 전화 번호
			- 가게 주소
			- 가게 소개
				※ 300글자 이내로 작성 할 수 있다.
			- 엽업 시간(평일, 주말 - open, breCk, close)
			- 휴무일
			- 테이블 수
		- 가게 등록을 할 경우, 가게 이름을 사용하여 중복 체크를 수행한다.
		  단, 가게 이름이 변경되지 않았다면 중복 체크를 하지 않는다.
		- 가게 수정이 완료되면 "가게 관리 - 가게 수정"으로 이동한다.

	- 상품 수정
		- 다음 항목을 수정한다.
			- 상품 이미지
			※ 상품을 300x300 크기로 표시한다.
			- 상품 카테고리
			- 상품 이름
			※ 상품 이름이 변경되면 해당 가게의 상품 이름을 중복 체크한다.
			- 상품 가격
			- 상품 소개

		※ 수정할 때, 해당 가게의 상품 이름을 중복 체크한다.

	- 상품 삭제
		- 선택된 상품을 삭제한다.

F - C - 08 : 예약 현황
	- 예약 조회
		- 예약 날짜를 년, 월을 선택하여 조회한다.
		※ 첫 조회는 해당 년, 월로 자동 선택된다.

	- 예약 조회 - 결과
		- 조회 결과는 달력 형태로 표시한다.
		- 각 일에는 예약 시간, 이메일을 표시한다.

	- 각 항목을 클릭하여 상세 내용을 확인할 수 있습니다.
		- 예약 상세 정보를 확인 할 수 있습니다.
			- 닉네임, 전화번호, 이메일 , 예약날짜, 인원수, 메뉴판, 간이 계산
		- 예약 수정
			- [수정] 버튼을 누르면 변경된 예약 정보로 갱신합니다.
				- 닉네임, 전화번호, 이메일 , 예약날짜, 인원수, 메뉴판, 간이 계산
				※ 예약 날짜는 현재 날짜보다 1일 이후로 설정이 가능하다.
				※ 예약 시간이 영업 시간에 포함되어야 예약을 할 수 있다.
			- 수정을 완료하면 예약 보기 화면으로 이동한다.
		- 예약 취소
			- [취소] 버튼을 누르면 해당 예약을 취소합니다.
			- 예약을 취소하면 예약 보기 화면으로 이동한다.

	- 예약 삭제
		- 예약 항목을 선택하고 "삭제" 버튼을 눌러 해당 예약 항목을 삭제한다.

F - C - 10 : 사용자 리뷰 관리
	- 사용자 리뷰 리스트를 표시한다.
	- 각 사용자 리뷰에 표시할 정보는 다음과 같다.
		- 작성자 : 사용자 닉네임을 사용하여 표시한다.
		- 작성일(년-월-일 시:분)
		- 내용
		- 평점
		- [수정]
		※ 다음과 같은 항목에 따라 해당 버튼의 표시 여부를 결정된다.
			- 해당 사용자 리뷰가 자신이 작성한 리뷰이면 해당 버튼이 표시된다.
		※ 수정 버튼을 누르면 해당 리뷰를 수정할 수 있는 화면으로 이동한다.
		- 사장님 댓글
			- 작성자 : "사장님"으로 표시한다.
			- 작성일(년-월-일 시:분)
			- 내용
		※ 사용자 리뷰에 사장님 댓글이 없는 경우 표시하지 않는다.
		- [사장님 댓글 달기]
			- 내용
			※ 내용은 300글자 이내로 제한한다.
			- 등록/수정 버튼
		※ 다음과 같은 항목에 따라 해당 버튼의 표시 여부를 결정된다.
			- 사용자 화면에서 해당 버튼이 표시되지 않는다.
		  	- 관리자 화면에서 해당 버튼이 표시된다.
			- 사장님 댓글이 달려 있는 경우는 표시되지 않는다.
		※ 해당 버튼을 누르면 해당 사용자 리뷰에 사장님 댓글을 달 수 있는 컴포넌트가 추가된다.
	- 리뷰 리스트는 페이징 처리한다.

F - C - 11 : 가게 카테고리 관리
	- 조회
		- 상위 카테고리, 하위 카테고리를 입력하여 검색한다.
	
	- 조회 - 결과
		- 테이블 형태로 표시하되 페이징 처리를 한다.
		
	- 등록
		- 다음 항목을 등록한다.
			- 상위 카테고리
			※ 기본 값은 "없음"으로 한다.
			- 하위 카테고리
			※ 상위 카테고리를 "없음"으로 선택할 경우, 하위 카테고리는 상위 카테고리로 한다.
		※ 등록할 때, 상위 카테고리의 하위 카테고리를 중복 체크한다.
	
	- 수정
		- 다음 항목을 수정한다.
			- 상위 카테고리
			※ 기본 값은 "없음"으로 한다.
			- 하위 카테고리
			※ 상위 카테고리를 "없음"으로 선택할 경우, 하위 카테고리는 상위 카테고리로 한다.
		※ 수정할 때, 상위 카테고리의 하위 카테고리를 중복 체크한다.
	
	- 삭제
		- 카테고리 항목을 선택하고 "삭제" 버튼을 눌러 해당 카테고리 항목을 삭제한다.
