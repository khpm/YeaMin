메인 화면
	1. 가게 정보
		- 가게 정보를 표시한다
		- 지도에 가게 주소를 표시한다.
	2. 통계
		- 상품 top 5 : 상품 카테고리(전체), 특정 기간(전체)에 예약된 상품의 top 5 상품을 상품 수와 비율을 표시한다.
	3. 메뉴판
		- 상품 카테고리 정보별 상품 정보를 표시한다.
		- 각 상품의 개수를 설정할 수 있으며 합계 금액에 반영한다.
		- 예약하기 버튼을 클릭하여 예약 화면으로 이동한다.
	4. 리뷰 & 댓글
		- 리뷰 정보와 댓글 정보를 표시합니다.
		- 리뷰
			- 사용자 : 리뷰를 등록할 수 있으며, 자신이 작성한 리뷰를 수정/삭제할 수 있다.
			- 관리자 : 모든 사용자의 리뷰를 삭제할 수 있다.
			※ 리뷰를 삭제할 때, 리뷰에 포함된 댓글도 함께 삭제한다.
		- 댓글
			- 관리자 : 댓글 유무에 따라 사용자 리뷰에 댓글 버튼이 비활성화/활성화 된다.
				- 댓글 무 : 활성화된 댓글 버튼을 통해 댓글을 등록할 수 있다.
				- 댓글 유 : 댓글을 수정할 수 있다.

예약 화면
	- 날짜를 선택하면 평일/주말에 의해 예약이 가능한 시간을 구한다.
	- 예약 가능한 시간을 선택하면 해당 시간에 예약한 인원 수와 예약 가능한 인원 수를 구한다.
	- 이전 화면에서 받은 합계 금액을 표시한다.
	
게시판 화면
	- 게시글의 정보와 사용자 정보를 표시한다.
	- 게시글을 등록할 수 있다.
		- 관리자 : 공지사항으로 사용하며 조회 시 게시글 상단에 표시한다.			 
	- 댓글을 등록할 수 있다.
	- 게시글과 댓글을 수정/삭제할 수 있다.
		- 사용자 : 자신이 작성한 게시글과 댓글을 수정/삭제할 수 있다.
		- 관리자 : 자신이 작성한 게시글과 댓글을 수정할 수 있으며, 모든 사용자의 게시글과 댓글을 삭제할 수 있다.
	- 게시글을 제목, 내용, 작성자, 제목+내용으로 검색할 수 있다.
	
같이예약 화면
	- 해당 기능은 사용자만 사용할 수 있다.
	- 그룹방 아이디 & 비밀번호 설정
		- 그룹방 아이디 부여 버튼을 통해 그룹방 아이디를 부여 받는다.
		- 부여 받은 그룹방 아이디의 비밀번호를 설정 할 수 있다.
	- 그룹방 입장
		- 그룹방 아이디와 비밀번호를 입력 후 그룹방 입장 버튼을 통해 입장한다.
	- 그룹방에 입장한 사용자는 실시간 채팅을 할 수 있다.
	- 달력을 공유하여 약속날짜를 정할 수 있으며, 해당 날짜의 예약가능한 시간을 볼 수 있다.
	- 다음 버튼을 통해 메뉴판을 볼 수 있다.
	- 메뉴판을 공유하여 메뉴를 정할 수 있으며, 합계 금액을 표시한다.
