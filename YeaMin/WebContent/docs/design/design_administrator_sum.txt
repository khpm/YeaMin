	※ 표시할 정보 : 아이디, 닉네임, 핸드폰 번호

	- 가게 정보를 입력하여 가게 정보를 저장합니다.
	※ 가게 정보 : 가게 이미지, 가게 이름, 가게 전화 번호, 가게 주소, 가게 소개
			, 영업 시간(평일, 주말 - open, break, close),
			  마지막 주문시간(평일, 주말 - last orders)

	- 상품 정보 : 상품 이미지, ★상품 카테고리, 상품 이름, 상품 가격, 상품 소개, 

★사용자 관리


* 예약 현황
	- 예약 날짜를 년, 월을 선택하여 조회한다.
	- 조회 결과는 달력 형태로 표시한다.
		- 각 일에는 예약 시간, 아이디를 표시한다.
	- 각 항목을 클릭하여 상세 내용을 확인할 수 있습니다.
		- 예약 상세 정보를 확인 할 수 있습니다.
			- 아이디, 닉네임, 전화번호, 이메일 , 예약날짜, 인원수, 메뉴판, 간이 계산
		- 예약 수정
			- [수정] 버튼을 누르면 변경된 예약 정보로 갱신합니다.
		- 예약 취소
			- [취소] 버튼을 누르면 해당 예약을 취소합니다.
* 예약 이력
	- 예약 날짜를 년, 월을 선택하여 조회한다.
	- 조회 결과는 달력 형태로 표시한다.
		- 각 일에는 예약 시간, 아이디를 표시한다.
	- 각 항목을 클릭하여 상세 내용을 확인할 수 있습니다.
		- 예약 상세 정보를 확인 할 수 있습니다.
			- 아이디, 닉네임, 전화번호, 이메일 , 예약날짜, 인원수, 메뉴판, 간이 계산

* 가계 통계
	- 리뷰수, 리뷰 남/녀 통계, 조회수, 월별 매출액, 예약 연령대, 예약한 TOP 5 상품, 
	  1인당 평균 소요 금액, 시간별 평균 예약 수(2시간 단위)

* 사용자 리뷰 관리
		- 사장님 댓글
			- 작성자, 작성일(년-월-일 시:분), 내용
		※ 사용자 리뷰에 사장님 댓글이 없는 경우 표시하지 않는다.
		- [사장님 댓글 달기]
			- 내용, 등록/수정 버튼
		※ 다음과 같은 항목에 따라 해당 버튼의 표시 여부를 결정된다.
			- 사용자 화면에서 해당 버튼이 표시되지 않는다.
		  	- 관리자 화면에서 해당 버튼이 표시된다.
			- 사장님 댓글이 달려 있는 경우는 표시되지 않는다.
		- 리뷰 리스트는 페이징 처리한다.
