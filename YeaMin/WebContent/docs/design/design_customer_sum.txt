* 로그인 화면
	- 아이디, 비밀번호를 입력하여 로그인을 시도한다.

* 회원가입 화면
	- 사용자 정보를 입력하여 회원가입을 합니다.
	※ 사용자 정보 : 아이디, 이메일, 비밀번호, 닉네임, 핸드폰 번호, 생년월일, 성별

* 가게 정보
	- 가게 정보를 표시한다.
		- 가게 이미지, 가게 이름, 가게 전화 번호, 가게 주소, 가게 소개
		- 영업 시간(평일, 주말 - open, break, close), 테이블 수,
		  마지막 주문시간(평일, 주말 - last order),
		- 예약수, 평균 별점 - 리뷰수
		- 지도
	- 메뉴판 정보를 표시한다.
	- 가계 통계를 표시한다.
	- 리뷰 리스트를 표시한다.
	- 예약하기 버튼을 눌러 예약할 수 있다.

* 메뉴판 정보
	- 메뉴판(상품 정보)를 표시한다.
		- 상품 이미지, 상품 카테고리, 상품 이름, 상품 가격, 상품 소개
		- 각 상품의 수량을 선택하여 간이 계산을 할 수 있어야 한다.
	- 상품 정보는 카드 컴포넌트를 사용하여 리스트 형태로 표시하며 페이징 처리를 한다.

* 가게 통계
	- 예약한 TOP 5 상품
	- 1인당 평균 소요 금액
	- 시간별 평균 예약 수(2시간 단위)

* 리뷰 리스트
	- 리뷰 쓰기
	- 리뷰 정보를 표시한다.
		- 닉네임, 작성일(년/월/일 시:분), 리뷰 내용, 평점
	- 리뷰 정보는 페이징 처리한다.
	- [수정]
	※ 다음과 같은 항목에 따라 해당 버튼의 표시 여부를 결정된다.
	- 해당 사용자 리뷰가 자신이 작성한 리뷰이면 해당 버튼이 표시된다.
	※ 수정 버튼을 누르면 해당 리뷰를 수정할 수 있는 화면으로 이동한다.

* 예약하기
	- 로그인을 해야 해당 기능을 사용할 수 있다.
	- step1 : 메뉴판(선택한 항목은 체크되어 있음), 간이 계산 결과
	- step2 : 예약 날짜, 예약 시간, 인원수, 전달사항

* 마이 페이지 - 회원 정보 수정
	- 사용자 정보를 입력하여 회원 정보를 수정합니다.
		- 사용자 정보 중 이메일은 수정할 수 없으며 표시합니다.
		- 사용자 정보 : 비밀번호, 닉네임, 핸드폰 번호, 생년월일, 성별
	  	※ 비밀번호, 닉네임, 핸드폰 번호, 생년월일, 성별은 필수 항목입니다.

* 마이 페이지 - 회원 탈퇴
	- 해당 사용자에 관련된 정보를 표시한다.
		- 리뷰수, 예약 이력 수, 즐겨찾기 등등...
	- 위에 명시된 항목은 "사용자 정보는 2년간 보관되며 다시 회원가입 할시에 연동됩니다."라고 표시한다.
	- 탈퇴한 회원은 따로 회원 목록에서 탈퇴한 회원 목록으로 옮긴다.

* 마이 페이지 - 예약 현황
	- 예약한 정보를 표시합니다.
		- 가게 이름, 가게 전화번호, 가게 주소, 예약 날짜(년-월-일 시:분)
	- 각 항목을 클릭하여 상세 내용을 확인할 수 있습니다.
		- 예약 상세 정보를 확인 할 수 있습니다.
			- 가게 이름, 가게 전화번호, 가게 주소,예약날짜, 인원수, 메뉴판, 간이 계산
		- [수정] 버튼을 누르면 변경된 예약 정보로 갱신합니다.
		- [취소] 버튼을 누르면 해당 예약을 취소합니다.

* 마이 페이지 - 예약 이력
	- 과거에 예약된 예약 정보를 표시합니다.
	- 표시할 정보는 다음과 같습니다.
		- 가게 이름, 가게 전화번호, 가게 주소, 예약 날짜(년-월-일 시:분)
	- 각 항목을 클릭하여 상세 내용을 확인할 수 있습니다.
		- 예약 상세 정보를 확인 할 수 있습니다.
			- 가게 이름, 가게 전화번호, 가게 주소,예약 날짜, 인원수, 메뉴판, 간이 계산
