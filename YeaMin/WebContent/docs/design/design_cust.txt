		- 로그아웃 버튼을 통해 로그아웃을 할 수 있으며 메인 화면으로 이동한다.(F - C - 02)
		- 마이 페이지 버튼을 통해 마이 페이지 화면으로 이동한다.(F - C - 12)

F - C - 03 : 로그인 화면
	- 이메일, 비밀번호를 입력하여 로그인을 시도한다.
		- 로그인에 실패할 경우, 다음과 같이 경우에 따라 메세지를 표시한다.
			- 이메일이 없는 경우, "입력하신 이메일/비밀번호가 잘못 되었습니다."라고 메시지 표시합니다.
			- 비밀번호가 일치하지 않는 경우, "입력하신 이메일/비밀번호가 잘못 되었습니다."라고 메시지 표시합니다.
		- 로그인에 성공할 경우, 로그인 화면에 이동되기 전 화면으로 이동한다.
		- 로그인에 성공할 경우, 확인 되지 않는 예약 건 수를 표시하며 "예약 확인"을 눌렀을 경우, 예약 현황 화면으로 이동한다.
	- 회원 가입 버튼을 통해 회원 가입 화면으로 이동 할 수 있다.(F - C - 04)

F - C - 04 : 회원가입 화면
	- 사용자 정보를 입력하여 회원가입을 합니다.
		- 사용자 정보 : 이메일, 비밀번호, 닉네임, 핸드폰 번호, 생년월일, 성별
		  ※ 이메일 양식에 어긋나면 표시합니다.
		  ※ 이메일, 비밀번호, 닉네임, 핸드폰 번호, 생년월일, 성별은 필수 항목입니다.
	- 닉네임을 사용하여 중복 체크를 합니다.
	  ※ 해당 항목을 수정할 경우, 중복 체크을 다시 진행합니다.
	- 이메일을 사용하여 중복 체크를 합니다.
	  ※ 해당 항목을 변경될 경우, 중복 체크와 이메일 인증을 다시 진행합니다.
	- 이메일 중복이 되지 않았으면 이메일을 사용하여 인증 절차를 진행합니다.
	- 회원 가입 버튼을 누르면 닉네임, 이메일을 사용하여 중복 체크를 합니다.
	  중복이 되었다면 다시 입력하도록 합니다.

F - C - 30 : 가게 검색 화면
	- "읍/면/동/리"를 입력 받아 검색할 지역을 선택합니다.
	- 가게 카테고리로 검색할 수 있다.
	- 가게 이름로 검색할 수 있다.
	- 상품 이름으로 검색할 수 있다.
	- 검색 버튼을 누르면 검색 조건에 해당하는 가게들을 표시한다.(F - C - 05)

F - C - 05 : 가게 리스트
	- 가게를 검색 할 수 있다.(F - C - 30)
	- 가게 정보를 표시한다.
		- 가게 이미지, 가게 이름, 가게 주소, 평균 별점(전체 리뷰수), 가게 소개, 예약수
	  	※ 가게 주소는 동까지 표시한다.
	  	※ 가게 소개는 20글자 이후는 ...을 붙여 줄인다.
	- 가게 정보는 카드 컴포넌트를 사용하여 리스트 형태로 표시하며 페이징 처리를 한다.
	- 가게 이름, 평균 별점 정렬, 리뷰수 정렬, 예약수로 정렬할 수 있다.
	  ※ 기본적으로 가게 이름으로 정렬한다.
	  ※ 모든 정렬은 내림차순으로 정렬한다.
	- 각 가게를 누르면 가게 정보를 확인할 수 있는 화면으로 이동한다.(F - C - 06)

F - C - 06 : 가게 정보
	- 즐겨찾기 버튼 눌러 즐겨찾기의 활성화/비활성화를 할 수 있습니다.
	  ※ 즐겨찾기 된 가게는 마이 페이지의 즐겨찾기 목록에서 확인 할 수 있습니다.(F - C - 16)
	- 가게 이미지, 상품 이미지를 슬라이드 형태로 표시한다.
	- 가게 정보를 표시한다.
		- 가게 이미지, 가게 이름, 가게 카테고리, 가게 전화 번호, 가게 주소, 가게 소개
		- 엽업 시간(평일, 주말 - open, break, close), 테이블 수, 휴무일
		- 예약수, 평균 별점 - 리뷰수
		- 지도 & 길찾기
	- 메뉴판 정보를 표시한다.(F - C - 07)
	- 가계 통계를 표시한다.(F - C - 08)
	- 리뷰 리스트를 표시한다.(F - C - 09)
	- 예약하기 버튼을 눌러 예약할 수 있다.(F - C - 10)

F - C - 07 : 메뉴판 정보
	- 메뉴판(상품 정보)를 표시한다.
		- 상품 이미지, 상품 이름, 상품 카테고리, 상품 가격, 상품 소개
		- 각 상품의 수량을 선택하여 간이 계산을 할 수 있어야 한다.
		※ 상품 이미지는 툴팁 형태로 표시한다.
	- 상품 정보는 카드 컴포넌트를 사용하여 리스트 형태로 표시하며 페이징 처리를 한다.
	- 간이 계산 결과
		- 상품 정보에서 간이 계산에 포함 시킨 항목을 계산하여 표시한다.

F - C - 08 : 가게 통계
	- 예약한 TOP 5 상품
	- 1인당 평균 소요 금액
	- 시간별 평균 예약 수(2시간 단위)

F - C - 09 : 리뷰 리스트
	- 리뷰 쓰기
		- 리뷰 내용을 입력 할 수 있다.
		※ 리뷰 내용은 300글자 이내로 작성한다.
		- 로그인을 해야 해당 기능을 사용할 수 있다.(F - C - 03)
		- 해당 가게에 예약 이력이 있는 사용자만 리뷰를 쓸 수 있다.
	- 리뷰 정보를 표시한다.
		- 닉네임, 작성일(년/월/일 시:분), 리뷰 내용, 평점
	- 리뷰 정보는 카드 컴포넌트를 사용하여 리스트 형태로 표시하며 페이징 처리한다.
	- 작성일, 별점수으로 정렬할 수 있다.
	※ 기본적으로 작성일로 정렬한다.
	※ 모든 정렬은 내림차순으로 정렬한다.

F - C - 10 : 예약하기
	- 로그인을 해야 해당 기능을 사용할 수 있다.(F - C - 03)
	- step1 : 메뉴판(선택한 항목은 체크되어 있음), 간이 계산 결과
	- step2 : 예약 날짜, 예약 시간, 인원수, 전달사항
	※ 예약 날짜는 현재 날짜 이후로만 설정이 가능하다.
	※ 예약 시간이 영업 시간에 포함되어야 예약을 할 수 있다.

F - C - 12 : 마이 페이지 - 회원 정보 수정
	- 사용자 정보를 입력하여 회원 정보를 수정합니다.
		- 사용자 정보 중 이메일은 수정할 수 없으며 표시합니다.
		- 사용자 정보 : 비밀번호, 닉네임, 핸드폰 번호, 생년월일, 성별
	  	※ 비밀번호, 닉네임, 핸드폰 번호, 생년월일, 성별은 필수 항목입니다.
	- 닉네임을 사용하여 중복 체크를 합니다.
	  ※ 해당 항목을 수정할 경우, 중복 체크을 다시 진행합니다.
	  ※ 기존에 사용하던 닉네임일 경우, 중복 체크를 하지 않습니다.
	- 회원 정보 수정 버튼을 누르면 닉네임을 사용하여 중복 체크를 합니다.
	  ※ 기존에 사용하던 닉네임일 경우, 중복 체크를 하지 않습니다.
	- 회원 정보 수정 시 회원 정보 수정 화면으로 이동합니다(F - C - 12)

F - C - 13 : 마이 페이지 - 회원 탈퇴
	- 해당 사용자에 관련된 정보를 표시한다.
		- 리뷰수, 예약 이력 수, 즐겨찾기 등등...
	- 위에 명시된 항목은 "사용자 정보는 2년간 보관되며 다시 회원가입 할시에 연동됩니다."라고 표시한다.
	- 탈퇴한 회원은 따로 회원 목록에서 탈퇴한 회원 목록으로 옮긴다.
	- 회원 탈퇴 시 메인 화면으로 이동합니다.(F - C - 02)

F - C - 14 : 마이 페이지 - 예약 현황
	- 예약한 정보를 표시합니다.
	- 표시할 정보는 다음과 같습니다.
		- 가게 이름
		- 가게 전화번호
		- 가게 주소
		- 예약 날짜(년-월-일 시:분)
	- 각 항목을 클릭하여 상세 내용을 확인할 수 있습니다.
		- 예약 상세 정보를 확인 할 수 있습니다.
			- 가게 이름, 가게 전화번호, 가게 주소,예약날짜, 인원수, 메뉴판, 간이 계산
		- [수정] 버튼을 누르면 변경된 예약 정보로 갱신합니다.
			- 가게 이름, 가게 전화번호, 가게 주소, 예약날짜, 인원수, 메뉴판, 간이 계산
			- 가게 이름, 가게 전화번호, 가게 주소는 변경 할 수 없으며 표시합니다.
			※ 예약 날짜는 현재 날짜보다 1일 이후로 설정이 가능하다.
			※ 예약 시간이 영업 시간에 포함되어야 예약을 할 수 있다.
		- [취소] 버튼을 누르면 해당 예약을 취소합니다.
			- 예약 시간이 현재 시간으로부터 2시간 이내이면 취소가 불가능 합니다.

F - C - 16 : 마이 페이지 - 즐겨찾기
